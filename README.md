#스케쥴러 샘플 프로젝트
## 목표
해당 스케쥴러 Rest API기반에 JPA 기반으로 TDD 방식을 접목하는 샘플형 프로젝트를 목표로 하며, 향후 프로젝트 진행시 바로바로 해당 소스에 필요한 부분을 뜯어내서 구성 가능하게 하는 것을 목표로 합니다.

## 스팩
- spring boot 2.4.5
- JPA
- MySQL [ schedulerdb ]
- hikari
